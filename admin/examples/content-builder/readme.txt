== Description ==

Content Builder is a Word press Plugin which lets you feel free to create any page layout that you could imagine, and it is simple as drag & drop.

This plugin will improve your website  functionality in few minutes and will save you a lot off time for managing content.

== Requirements ==

- jQuery at least 1.7.2, jQuery UI at least 1.8.2
- PHP 5 with GD library;
- Permission to `/wp-content/uploads/`

== Installation ==

1. Upload the `content-builder` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Got to `Options`, `Content Builder` to set it up
